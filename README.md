# MagnonCRM

### Main technology stack:
- Java 11+
- Spring boot 2.4.2
- Gradle 6.8.1 (Kotlin DSL)
- Vue.js 2.x
- Webpack 4.x

### Preparation

- Install Java 11 or higher version
- Clone Repository

        git clone https://github.com/Dreadblade-dev/MagnonCRM
        cd MagnonCRM

### Usage

- Run application

        gradlew bootRun

- Or run frontend

        cd frontend
        npm run serve
